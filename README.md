# C# Sample Project

This project demonstrates several C# concepts and techniques:

- Working with nested lists and using LINQ `Select`.
- Using `Join` to match users with their orders.
- Implementing a private constructor and a static factory method (`Addition`).
- Using `ReadOnlySpan<char>` for zero-allocation string operations.

Ideal for beginners, this project helps you learn basic LINQ, class design, and memory-efficient string handling in C#.
